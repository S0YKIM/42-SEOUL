⭐isalpha()
===================

SYNTAX
------
        int					ft_isalpha(int c)

DESCRIPTION
----------- 

* (unsigned char 로 변환된) `c` 가 알파벳인지 확인
* 'a'부터 'z'까지
* 'A'부터 'Z'까지

⭐isdigit()
================

SYNTAX
------
    int					ft_isdigit(int c)

DESCRIPTION
-----------

* (unsigned char 로 변환된) `c` 가 숫자인지 확인
